# Energy Management Skill Handoff

Treat energy-management as a planned skill, not an installable artifact yet. Start with an issue handoff that captures target users, non-medical safety boundaries, cadence assumptions, and examples of weekly planning prompts.

Promotion criteria: the handoff should define trigger phrases, references to load, outputs to produce, validation prompts, and a small benchmark set before any portable skill package is created.

The first implementation pass should stay private-first, avoid medical claims, and document escalation language for cases where the user needs professional health support.
