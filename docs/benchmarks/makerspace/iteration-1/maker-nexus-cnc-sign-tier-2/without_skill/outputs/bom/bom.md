## Vendor notes

- **Ocooch Hardwoods (Wisconsin)** - long-time mail-order Baltic birch
  source, ships flat in protective cardboard. [verify current pricing]
- **Forest Plywood (Los Angeles)** - West Coast option, can will-call
  full sheets if you have the truck for it.
- **KJP Select Hardwoods (Toronto)** - high-quality but ships from
  Canada; rarely the cheapest option for a Sunnyvale-based build.
- **Amana Tool** - direct from amanatool.com or via ToolsToday. Genuine
  Amana bits, not the gray-market clones.
- **IDC Woodcraft** - well-regarded for hobbyist-priced bits.
- **Osmo** - Polyx-Oil distributed by Woodcraft, Lee Valley, Rockler,
  and Amazon. Buy the 0.125 L can - the 0.75 L is too much for one
  sign.

## Substitutes if a part is unavailable

| Original | Acceptable substitute | Trade-off |
|---|---|---|
| 1/4" Baltic birch | 1/4" maple or birch ply (B/BB grade) | Slightly worse face quality |
| 60 deg V-bit | 90 deg V-bit | Shallower carve, less drama |
| 1/8" upcut | 1/4" upcut | Bigger inside corners (0.125 in radius) |
| Osmo Polyx-Oil | Tried & True Original (linseed/wax) | Slower cure, similar look |
| Carpet tape | Double-stick painter's tape sandwich | Cheaper, weaker hold |
