# Tools and Settings

## Machine settings — Walking-foot industrial (Juki DU-1181N or similar)

| Setting | Value | Notes |
|---|---|---|
| Needle | Schmetz 110/18 jeans/denim or 100/16 leather (system 135x17) | 100/16 for lining-only seams; 110/18 for canvas |
| Top thread | Tex 40 / V-46 polyester (Mara 70 equivalent) | Match color exactly for stitches that show |
| Bobbin thread | Same Tex 40 OR Tex 30 in matching color | Tex 30 in bobbin reduces bulk on intersections |
| Stitch length | 3.5 mm (about 7 SPI) for canvas; 2.5 mm for lining | Longer stitches read as "intentional" on heavy fabric |
| Foot pressure | Medium-light | Walking foot does the work; over-pressing scars the wax |
| Tension (top) | Slightly looser than default | Heavier thread + thicker fabric needs the loop to set deeper |
| Reverse | Use 3-4 stitches at start/end | Lock-stitch button if available |
| Speed (servo) | Max 1500 RPM until comfortable | These can run 3000+; no need |

## Machine settings — Domestic backup (if not yet certified)

| Setting | Value |
|---|---|
| Needle | Schmetz 110/18 jeans (system 130/705H) |
| Foot | Walking foot attachment OR Teflon foot OR roller foot |
| Top thread | Tex 40 polyester |
| Stitch length | 3.5-4.0 mm |
| Tension | Increase top tension by 0.5-1.0 from default |
| Speed | Slow and steady — no speed competition |

## Hand tools
- **Wonder clips** — at least 30. You'll use them everywhere.
- **Hera marker or tailor's chalk** — never friction pens, never pencil
- **Sharp rotary cutter** — replace blade before starting
- **Long quilting ruler** (24 in) and self-healing mat
- **Sharp scissors** for trimming threads (not for cutting waxed canvas — rotary only)
- **Seam ripper** (you will need it)
- **Awl or stiletto** for guiding fabric near the needle on thick stacks (keeps fingers safe)
- **Loop turner or chopstick** for turning corners

## Pressing tools
- **Pressing cloth** (silk organza or cotton muslin) — required for waxed canvas
- **Iron on LOW heat** or hair dryer on warm — never direct iron on waxed canvas right side
- **Wooden clapper or seam roller** — sets seams without heat

## Why these specific choices

**Why Tex 40 thread?** It's the sweet spot for heavy canvas. Tex 30 (general all-purpose) breaks under stress at strap anchors. Tex 70 (upholstery) is overkill and won't fit through a 110/18 needle eye cleanly.

**Why a 110/18 jeans needle and not a 110/18 leather?** Leather needles have a chisel point that cuts the fabric — fine for leather, but on woven canvas it severs threads and weakens the seam over time. Jeans/denim needles have a sharp round point that pierces between threads.

**Why no pins?** Waxed canvas does not heal — every pin hole is permanent. Clips only.

**Why a pressing cloth?** Direct iron contact remelts the wax and transfers it to your iron's soleplate. Once that happens you'll spend 30 minutes scraping wax off your iron and the next person to use it will be unhappy.
