# Changelog

## 1.0.0 - 2026-05-08

Initial release. Weekly-to-biweekly disk recovery for multi-worktree dev setups: `cargo clean`, merged-branch pruning, npm/pnpm cache cleanup, optional Docker prune, and WSL2 VHDX shrink guidance. Defaults to `--dry-run`; destructive operations require `--apply`.
