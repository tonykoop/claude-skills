# Sensor Capture Checklist

Use one copy per configuration or session.

## Microphone

- [ ] Microphone model recorded.
- [ ] Mic distance measured from the same reference point each run.
- [ ] Mic angle/orientation recorded.
- [ ] Input gain recorded in dB or interface knob position.
- [ ] Limiter, compression, noise reduction, and automatic gain state recorded.
- [ ] Stand, cable, and device are outside the string/spring release path.

## Contact Sensor

- [ ] Contact mic or piezo model recorded.
- [ ] Mounting location photographed.
- [ ] Mounting material recorded: tape, putty, clamp, wax, adhesive, or other.
- [ ] Cable strain relief added.
- [ ] Sensor mass is noted if it may affect the measured part.

## Environment

- [ ] Room or outdoor site recorded.
- [ ] Temperature recorded.
- [ ] Relative humidity recorded.
- [ ] Background noise noted.
- [ ] Floor/table/suspension surface recorded.
- [ ] Nearby reflective surfaces or damping materials noted.

## Excitation

- [ ] Excitation method recorded: pluck, strike, bow, tap, scrape, driver, or other.
- [ ] Tool recorded: finger, pick, mallet, bow, shaker, solenoid, or other.
- [ ] Force proxy recorded where possible: drop height, pick mark, driver setting, or repeatable hand cue.
- [ ] Repeat count recorded.
- [ ] Operator recorded.

## Subjective Descriptors

- [ ] Attack.
- [ ] Bloom.
- [ ] Buzz or rattle.
- [ ] Pitch stability.
- [ ] Sustain.
- [ ] Harshness or sweetness.
- [ ] Perceived coupling between string, spring, membrane, and frame.
