# Stored-Energy Safety Checklist

Use this checklist before any string/spring/membrane rig run.

## Before Assembly

- [ ] Maximum string tension estimated or measured.
- [ ] Maximum spring extension estimated or measured.
- [ ] Anchor, clamp, screw, knot, crimp, and fastener ratings exceed expected load.
- [ ] Brittle parts, sharp edges, and pinch points identified.
- [ ] Secondary restraint planned for every spring or string that could release toward a person.

## Before Each Run

- [ ] Strings inspected for fraying, kinks, corrosion, or sharp bends.
- [ ] Springs inspected for deformation, cracks, or damaged hooks.
- [ ] Membrane inspected for tears, slipping, or uneven tension.
- [ ] Bridge and frame inspected for cracking, lifting, or loose fasteners.
- [ ] No face, hand, camera, or microphone is in the release path.
- [ ] Eye protection is used when tensioned parts are exposed.

## Stop Rules

Stop immediately if any of these occur:

- [ ] Anchor slips or creeps.
- [ ] Frame, bridge, membrane ring, or clamp cracks.
- [ ] String frays or unwinds.
- [ ] Spring deforms or heats unexpectedly.
- [ ] Rig oscillates uncontrollably.
- [ ] A component contacts the secondary restraint.
- [ ] Operator cannot repeat the excitation safely.

## After Run

- [ ] Tension released or reduced before unattended storage.
- [ ] Any damage photographed and logged.
- [ ] Failed parts tagged and removed from reuse.
- [ ] Safety notes copied into the measurement log.
