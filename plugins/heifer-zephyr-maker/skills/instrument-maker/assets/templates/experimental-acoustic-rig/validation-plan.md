# Validation Plan

## Stage 1: Coupon

- [ ] Measure one element by itself.
- [ ] Record baseline response in `measurement-log-template.csv`.
- [ ] Confirm safety bounds in `variable-matrix.csv`.

## Stage 2: Single Subsystem

- [ ] Combine two elements, such as string plus bridge or spring plus membrane.
- [ ] Change one independent variable at a time.
- [ ] Repeat each run at least three times.

## Stage 3: Coupled Bench Rig

- [ ] Add the full coupled path.
- [ ] Keep sensor positions fixed during each variable sweep.
- [ ] Capture audio/contact/environment data for every run.

## Stage 4: Alpha Instrument Gate

- [ ] Dominant variables are identified.
- [ ] Repeatability is good enough to justify ergonomic layout decisions.
- [ ] Stored-energy safety checklist passes without exceptions.
- [ ] Remaining unknowns are documented in `risks.md`.
