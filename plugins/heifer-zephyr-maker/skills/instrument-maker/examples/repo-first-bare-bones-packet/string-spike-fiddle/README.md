# String / Spike-Fiddle Bare-Bones Variant

Use this compact variant for erhu, huqin, or similar spike-fiddle starter repos
when the public repo needs a first packet but measured setup geometry is not
ready.

Status language:

- Current status: `bare-bones readiness packet`
- Fabrication authority: `not build-ready`
- CAD/DXF status: `future authority unless measured geometry already exists`
- Measurement status: qianjin-to-bridge, bridge, qianjin, resonator/body,
  membrane/soundboard interface, neck/spike, and peg/string path are
  `measurement-required`

Do not use total string length as the active scale length. For erhu-like
starters, record `380-420 mm` only as a first-pass qianjin-to-bridge setup
range with status `measurement-required` or `assumption`.

Do not create cut-ready DXF/CAD from this example. The future authority should
be a measured template, reviewed drawing, DXF/CAD file, or design table named
in the packet's drawing brief.
