# Style Profiles

Use these profiles to translate inspiration into process constraints. Do not
clone a living artist's exact style. Borrow design principles such as layer
logic, relief depth, registration, density, negative space, and material
contrast, then make an original piece for the user's subject.

## Gabriel Schama-Inspired Relief

Useful for stacked plywood wall art, portraits, mandala-like geometry, flowing
vector ribbons, and topographic surface depth.

- Favor nested vector paths, flowing swirls, portrait/symbol overlays, and
  repeating contours.
- Use medium-to-high layer counts, often 4-12 for practical shop work.
- Let shadows come from real layer depth, not only engraved darkness.
- Track grain direction, glue-up order, backer visibility, and edge finishing.
- Output should emphasize layer maps, stack order, alignment dowels, and a
  sanding/finish plan.

Reference link: https://www.gabrielschama.com/

## Martin Tomsky-Inspired Narrative Relief

Useful for storybook panels, illustrative scenes, silhouettes, landscapes,
keepsakes, jewelry, and decorative wall pieces.

- Favor readable silhouettes, character or scene hierarchy, and contour-line
  storytelling.
- Use stained or naturally varied plywood layers to create color separation.
- Keep delicate details connected to larger structures.
- Avoid over-posterizing; expressive linework matters more than photoreal value.
- Output should include scene layers, outline/detail hierarchy, and assembly
  sequencing.

Reference link: https://www.martintomsky.com/about

## Broinwood/Nik-Inspired Inlay

Useful for CNC inlay, laser-cut veneer inlay, contrasting wood compositions,
cutting-board style motifs, and clean vector toolpath families.

- Separate pocket geometry from insert geometry.
- Keep vectors clean, closed, artifact-free, and assigned to explicit layers.
- Account for bit angle, cutter diameter, inlay clearance, glue squeeze, and
  sanding reveal.
- Choose contrasting woods or veneers with clear value separation.
- Output should include pocket/insert layer names, toolchain notes, and warnings
  that final parameters depend on tooling and material tests.

Reference link: https://broinwood.com/pages/about-us

## Eric Standley-Inspired Paper Architecture

Useful for deep paper relief, colored cardstock stacks, architectural lattice,
ornament, stained-glass-like negative space, and light/shadow installations.

- Favor many thin layers, frame-to-frame variation, apertures, and nested rooms.
- Treat negative space as the primary medium.
- Require stronger bridge checks because paper islands and lacework fail easily.
- Plan spacers, compression, adhesive method, humidity behavior, and lighting.
- Output should include layer count, paper weight, color order, stack alignment,
  and handling precautions.

Reference link: https://www.eric-standley.com/

## Multi-Layer Spray Stencil

Useful for flat art, posters, canvas wraps, panels, cajon faces, murals, and
repeatable paint editions.

- Start with value separation: base shadow, one or more midtones, highlight,
  then optional sharp detail.
- Use Mylar for repeatability and cardstock for quick/low-cost experiments.
- Add bridges to prevent islands from falling out.
- Make registration pins or marks non-negotiable.
- Output should include layer order, paint colors, overspray control, dry times,
  and an alignment jig.

## Hybrid Voice

When the user brings a dream, memory, or story, use the style profiles as
fabrication languages:

- Schama-like: "memory as carved depth."
- Tomsky-like: "story as layered illustration."
- Broinwood-like: "symbol as precise contrast inlay."
- Standley-like: "absence and light as architecture."
- Spray stencil: "instant graphic impact through color passes."
