# Classes — example-space

Class catalog. Each class section anchors to the YAML `id` in
`profile.yaml::classes`.

## `#class-shop-orientation`

1-hour shop orientation. Required before any other class. Covers
shop layout, emergency procedures, dust collection, PPE
expectations. Free for members; required as part of onboarding.

## `#class-cnc-intro`

4 hours, $TBD. Hands-on intro to CNC routing covering:
- Vectric / Fusion CAM workflow
- Tool selection and feeds/speeds for plywood and hardwood
- Workholding (vacuum table + tabs)
- Setting work zero, running first job
- What to do when the cut doesn't look right

Grants: `#cnc-router-cert`. Taught by: TBD.
