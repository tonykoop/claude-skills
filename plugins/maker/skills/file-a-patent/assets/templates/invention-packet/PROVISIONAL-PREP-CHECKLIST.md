This checklist is for preparation and attorney review. It is not a filing instruction.

## Core Materials

- [ ] Written description covers the preferred embodiment.
- [ ] Written description covers important alternate embodiments.
- [ ] Necessary drawings are listed.
- [ ] Figure labels and part names are consistent.
- [ ] Inventor list is drafted for attorney review.
- [ ] Public-disclosure timeline is complete enough for counsel.
- [ ] Prior-art/background search log is started.
- [ ] Best mode/preferred build is described.
- [ ] Experimental data and validation status are documented.
- [ ] Employer/client/provenance issues are flagged.
- [ ] Current USPTO rules and fees have been verified from official sources.
