# Official Sources

Use this reference when current USPTO or IP-process facts matter. Verify these sources again before relying on fees, deadlines, forms, or filing mechanics.

Last checked: 2026-05-07.

## USPTO Provisional Applications

- Source: https://www.uspto.gov/patents-getting-started/patent-basics/types-patent-applications/provisional-application-patent
- Key working notes:
  - A provisional application can establish an early U.S. filing date for a later nonprovisional application.
  - It must include a written description sufficient under 35 U.S.C. 112(a).
  - Drawings needed to understand the invention should be filed with the provisional; necessary drawings cannot be added later as new matter.
  - It must name all inventors.
  - It needs a cover sheet or equivalent application data and the filing fee.
  - It is not examined and does not mature into a granted patent by itself.
  - A corresponding nonprovisional generally must be filed within 12 months to preserve benefit.
  - Provisional applications cannot be filed for design inventions.
  - U.S. inventor grace-period rules may not preserve foreign rights after public disclosure.

## USPTO Design Patents

- Source: https://www.uspto.gov/patents-getting-started/patent-basics/types-patent-applications/design-patent-application-guide
- Key working notes:
  - Design patents concern visual ornamental characteristics embodied in or applied to an article of manufacture.
  - Utility patents generally protect how something is used and works; design patents protect how it looks.
  - A design application generally has one claim directed to the ornamental design as shown, or as shown and described.

## USPTO Trade Secrets

- Source: https://www.uspto.gov/ip-policy/trade-secret-policy
- Use when deciding whether to document something as a possible trade-secret candidate. Trade-secret value depends on secrecy measures; do not publish or over-disclose those details.

## Creative Commons CC BY 4.0

- Source: https://creativecommons.org/licenses/by/4.0/legalcode
- Working note: CC BY 4.0 is a broad public copyright/database license, is designed to be irrevocable for granted rights if terms are followed, and does not license patent or trademark rights. Still, public release can create patent-disclosure issues.

## MIT License

- Source: https://opensource.org/license/mit
- Working note: MIT is a permissive software/documentation copyright license. It is usually not a patent strategy document; do not rely on it to preserve secrecy or patent filing options.
