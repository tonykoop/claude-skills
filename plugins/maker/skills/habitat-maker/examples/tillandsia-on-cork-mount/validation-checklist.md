# Validation checklist — Tillandsia on cork-bark mount

Each item is a pass/fail gate, not a suggestion. Gate ids match the
[`tillandsia-on-bark`](../../references/tillandsia-on-bark.md) reference.

| Gate | Pass condition | Result |
|---|---|---|
| `till-no-copper` | No copper wire/fasteners/copper-treated wood contacts any plant. | [ ] |
| `till-removable` | Every plant lifts out, or the slab lifts off the wall, for soaking. | [ ] |
| `till-no-crown-glue` | No adhesive on the central crown/growing point (none used here). | [ ] |
| `till-drainage` | Orientation sheds water; no cupped pocket traps water at a base. | [ ] |
| `till-light-note` | Packet states the species light need (bright indirect). | [ ] |
| `till-hanger-mass` | Hanger rated for wet mass of cork + soaked plants. | [ ] |
| `habitat-substrate-safe` | No pressure-treated/resinous/painted wood in contact zone. | [ ] |

Fail remedies: see the reference's gate table. Any fail blocks the packet until
remedied; record evidence (photo or note) for each pass in `agent-record.md`.
