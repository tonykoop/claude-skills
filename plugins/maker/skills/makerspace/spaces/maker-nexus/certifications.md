# Certifications — Maker Nexus

Maker Nexus uses tool-area certifications. Some areas require a
single class; others have a multi-step path (orientation → area-specific
intro → advanced classes for complex tools).

## How certs work here

Certs are tracked in the member system. Once you complete a class,
the instructor logs it; the system flags you as cleared for the
relevant tools. When in doubt about your status, check the member
portal.

## `#shop-safety-cert`

The orientation everyone takes first. 1 hour. Covers shop layout,
emergency, dust collection, and PPE expectations.
- Granted by: `#class-shop-orientation`
- Required before any other cert

## `#woodshop-cert`

Tablesaw, miter saw, jointer, planer, drum sander, drill press,
bandsaw, router table.
- Granted by: `#class-woodshop-intro`
- Refresher: TBD

## `#cnc-cert`

CNC router area. Covers Vectric workflow (or Fusion CAM), tool
selection, workholding, work-zero, first-job procedures.
- Granted by: `#class-cnc-intro`

## `#laser-cert`

Laser cutter area.
- Granted by: `#class-laser-intro`

## `#3dp-fdm-cert` and `#3dp-resin-cert`

Two separate certs — FDM and resin printers have different safety
profiles.
- Granted by: `#class-3dp-fdm` and `#class-3dp-resin` respectively

## `#metalshop-cert`

Mills, lathes, welders, plasma. Larger area; some shops break this
into sub-certs (lathe-only, mill-only, welding-only) — verify the
current Maker Nexus structure.
- Granted by: `#class-metalshop-intro`

## `#textile-cert`

Sewing, sergers, embroidery.
- Granted by: `#class-textile-intro`

## What if your cert isn't current?

When the build packet calls for a tool you aren't cleared on, the
shop-planner specialist will:
1. Note the cert gap explicitly.
2. Surface the next class date if known.
3. Propose a workaround using cleared tools.
4. Add "get <cert>" to the prep checklist if the workaround isn't
   practical.

Don't run a tool without the cert. The cert system exists so that
people who haven't been trained don't get hurt.
