## Assumptions That Must Be Tested Or Replaced

1. Material thickness equals tabulated gauge thickness (it usually doesn't
   exactly). Measure each sheet before any flat-pattern release.
2. Inside bend radius equals material thickness. Replace with measured
   brake punch radius if larger.
3. K-factor 0.44 across all bends. Replace with measured coupon value.
4. Plasma kerf 0.080 in. Replace with measured kerf for the actual cut
   settings.
5. Box weight estimates use planning flat-pattern area, not measured
   nest area. Regenerate when SolidWorks Flat Pattern is rebuilt.
6. Stack envelope adds 2 * `Stack_Rim_Height` per box-to-box interface.
   If the rim sits inside the shoulder rather than stacking on top, this
   number should change. Decide and bake into the MLP.
7. Caster bolt pattern 2.5 in. Replace with the actual purchased caster's
   bolt pattern.
8. Hinge geometry uses a generic piano hinge slot pattern. Replace with
   the actual purchased hinge's leaf and knuckle dimensions.
9. Stack-coupling latches are assumed to mount in the same zones as
   tub-to-lid latches. Verify with hardware in hand.
