## Headline Design Choices
- **5052-H32 aluminum** at 1.6 mm — deforms instead of shattering. The "forgiving aluminum" the user asked for.
- **Single-piece folded shell** with six bends — no welding, repairable in a vise.
- **15-degree wedge** front, **180-degree wheel guards** — addresses the two most common beetleweight threats (horizontal spinners and lifters).
- **Removable arming link** — physical, kid-safe power disconnect.
- **M3 button-head fasteners throughout** — one hex key for the whole bot.
- **45-second top-deck removal** — meets tournament turnaround.

## Headline Numbers
- Target weight: 1290 g (95% of 1361 g limit, ~74 g headroom budgeted)
- Estimated build cost: $315 (excluding transmitter and tournament fees)
- Build time: 12-16 hours across 4-5 sessions
- Top speed: 2.1 m/s
