# learn-to-play / {{display_name}}

A starter songbook for the **{{display_name}}** you built from
[`{{build_repo}}`](https://github.com/{{build_repo}}). Generated by the
[`sheet-music`](https://github.com/tonykoop/sheet-music) skill.

## How to use this folder

Every tune has multiple files. The canonical source for each is
`tune.abc` — everything else is generated from it.

| File | Purpose |
|---|---|
| `tune.abc` | Canonical notation in ABC format (plain text) |
| `tune.ly` | Engraved notation in LilyPond (renderable to PDF/SVG) |
| `tune.musicxml` | Notation for Ableton, Logic, MuseScore, Sibelius, etc. |
| `tune.mid` | Playable MIDI (open in any media player) |
| `tune.wav` | Rendered audio (only if soundfont was available) |
| `tune.svg` | Engraved notation as a single-page SVG |
| `tune-fingering.svg` | Fingering chart for every distinct pitch |
| `notes.md` | Source / provenance, range checks, practice tips |
| `ableton-prompt.md` | Optional — paste into Claude+Ableton for a backing track |

## Folder map

- `00-warmup-scales/` — range walk and intervals to warm up.
- `01-easy/` — three beginner public-domain tunes.
- `02-intermediate/` — two tunes with simple ornaments.
- `03-original/` — one Heifer Zephyr original written for this build.
- `fingering-charts.svg` — combined chart covering the instrument's full range.
- `songsheet.pdf` — printable one-page summary.

## Suggested practice path

1. Run the warmup scales in `00-warmup-scales/` for 5 minutes.
2. Pick one tune from `01-easy/`. Listen to its `tune.wav` (or open
   `tune.mid` in any media player) until you can hum the melody.
3. Open the `notes.md` for that tune — read the range check and the
   practice tip.
4. Play the first phrase slowly with the fingering chart open.
5. When `01-easy/` feels comfortable, move to `02-intermediate/`.
6. Save `03-original/` for when you're ready to learn something new
   that wasn't written before. It's yours to perform — MIT licensed.

## Re-generating

If you change a `tune.abc` (transposing, ornamenting, fixing a typo),
regenerate the derived files:

```bash
# from the sheet-music repo root:
python scripts/render_pipeline.py \
  --tune <tune.abc path> \
  --instrument {{instrument_id}} \
  --out <output dir>
```
