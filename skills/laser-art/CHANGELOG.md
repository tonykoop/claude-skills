# Changelog

## 0.1.0 - 2026-05-11

- Added the first canonical `laser-art` skill package with top-level
  `version` and `last-updated` frontmatter.
- Documented laser artwork boundaries: generated images are concept/reference
  inputs, while reviewed SVG, DXF, CAD, templates, measured drawings, and test
  coupons remain fabrication authority.
- Added install/version guidance: v0.1.0 does not provide a `laser-art`
  executable or `laser-art --version`; use `skills-meta` for version checks
  until a real command shim exists.
