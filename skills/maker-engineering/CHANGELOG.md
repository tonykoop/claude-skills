# Changelog — maker-engineering

## v1.1.0 — 2026-05-10

- Added `references/human-carrying-safety-gate.md` — required safety gate
  for kayaks, boats, paddleboards, bike frames, treehouses, climbing rigs,
  ladders, swings, lift platforms, child seats, and any artifact that
  carries a person, floats with a person aboard, or sits above a person.
- The gate enumerates eight required sub-sections (intended environment,
  exclusions, anthropometric envelope, load cases and safety factor,
  prototype-vs-final boundary, staged validation gates, assisted
  first-use protocol, re-validation triggers) plus a verbatim
  "not a certification" clause.
- SKILL.md: added Safety Gate mode/section; new trigger phrases for
  human-carrying objects; new output rule mandating the gate's eight
  sub-sections plus the certification-disclaimer clause.
- routing-decision-tree.md: added §1a Safety-Critical Object Check;
  added kayak hybrid example with gate-first handoff order.
- doe-template.md: added cross-link to the safety gate.
